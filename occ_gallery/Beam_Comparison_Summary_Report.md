## 🚨 **「応力が低い=性能が良い」は危険な誤解**

### **なぜ曲率最適化はりの低応力は性能の良さではないのか**

1. **物理法則との矛盾**:
   ```
   応力 σ = M×c/I = (荷重による曲げモーメント)×距離/断面二次モーメント
   
   同じ荷重下では：
   - 実際はり: σ = 実際のM×c/I （物理法則通り）
   - 最適化はり: σ ≈ 0 （荷重無視のため非物理的）
   ```

2. **荷重抵抗メカニズムの欠如**:
   - **実際はり**: 荷重→モーメント→応力→抵抗のチェーンが確立
   - **最適化はり**: 荷重を受ける設計になっていない（美的形状のみ）

3. **構造安全性の根本的欠陥**:
   - **許容応力度**: 実際はりは材料強度に基づく安全設計
   - **最適化はり**: 荷重時の応力状態が予測不能

### **実用化時の致命的問題**

| 問題カテゴリ | 具体的リスク | 発生確率 |
|-------------|-------------|---------|
| **予期しない破壊** | 設計荷重以下での突然破壊 | 高 |
| **変形制御不能** | 使用限界状態の予測不可 | 高 |
| **動的応答異常** | 共振・不安定振動 | 中 |
| **疲労寿命短縮** | 想定外応力集中での亀裂進展 | 高 |
| **法規適合困難** | 建築基準法等の構造計算に適用不可 | 確実 |

### **正しい性能評価**

| 真の高性能はり | 特徴 |
|---------------|------|
| **適切な応力分布** | 荷重に応じた予測可能な応力 |
| **十分な安全余裕** | 許容応力度以下での設計 |
| **変形制御** | 使用限界の明確な管理 |
| **製造・施工可能性** | 実現可能な形状・寸法 |
| **経済性** | コスト効率的な材料使用 |

**結論**: 曲率最適化はりの「低応力」は**性能の良さではなく、荷重を考慮していない証拠**です。真の高性能とは、**予測可能で制御された応力状態**を実現することです。
