# 修正版：はり形状最適化vs従来形状の性能比較レポート

## 概要

FairCurveの曲率最適化理論に基づく変断面はりと、従来の一定断面はりの性能を**同一荷重条件下で**比較分析しました。先の計算で発見された単位系エラー（変位が数千km）を完全に修正し、SI単位系で統一した工学的に正しい計算を実装しました。

## 修正した主要な問題点

### 1. 単位系の混乱
- **旧版**: `length=1000.0` (mm) + `E=210e9` (Pa) + `I=8.33e-6` (m⁴) 
- **修正版**: 全計算をSI単位系 [m, N, Pa] で統一

### 2. 非現実的な計算結果
- **旧版**: 1mはりの変位が7,400km（物理的に不可能）
- **修正版**: 15～24mm（工学的に妥当）

### 3. 断面特性の不整合  
- **旧版**: 断面寸法と断面二次モーメントが矛盾
- **修正版**: 50×10mm断面から正しく計算（I = 4.17×10⁻⁹ m⁴）

## 解析条件

### はり仕様
- **寸法**: 長さ 1.0m × 幅 50mm × 高さ 10mm
- **材料**: 鋼材 (E = 210 GPa, 降伏強度 250 MPa)
- **支持条件**: 両端単純支持
- **細長比**: L/h = 100（はり理論適用範囲内）

### 荷重ケース
1. **等分布荷重**: 1000 N/m
2. **中央集中荷重**: 1000 N

## 最適化原理

### 等応力設計コンセプト
最適化形状は以下の工学原理に基づきます：

```
I(x) ∝ M(x)  (断面二次モーメント ∝ 曲げモーメント)
```

- **高モーメント部**: 断面を大きくして剛性向上
- **低モーメント部**: 断面を小さくして材料効率化
- **結果**: 応力分布の均一化と構造効率の最大化

### 断面変化パターン
- **断面変化比**: 0.5～1.5倍（モーメント分布に適応）
- **材料使用量**: 同一と仮定
- **応力平滑化**: 局所的応力集中の緩和
